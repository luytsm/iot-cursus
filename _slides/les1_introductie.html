LoRa

  • Radio(PHY) technologie van Semtech
  • LoRa Alliance: LoRaWAN standarisatie
  • LoRa netwerk kan door iedereen worden opgezet
  • Licensie voor chipmakers
  • Gepushed in België door Proximus

LoRa

  • 868 MHz (EU), 900 MHz(US) (433 Mhz Supported
  • Wideband (1,8 kHz - 500 kHz)
  • Chirp Spread Spectrum
  • Random Frequency and Time Division Multiple Access
  • Kleine Payload (2 - 255 bytes)
  • Bi directioneel capaciteiten / device class (A, B, C)

SigFox

  • LPWA Network Operator met Partners
  • 1 SigFox netwerk / land
  • Gepushed in België door Engie

Sigfox

  • 868 MHz (EU), 900 MHz(US) (433 Mhz Supported
  • Ultranarrowband (100 - 600 Hz)
  • Random Frequency and Time Division Multiple Access
  • Zeer kleine payload 12 Bytes
