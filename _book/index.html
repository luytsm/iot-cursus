Introductie

Voor het project vak Internet of Things moet je een hardware project realiseren. De focus ligt hier op het draadloos versturen van data die afkomstig van sensoren. Het kan ook de andere richting uitgaan, door het aansturen van actuatoren.

De project werking is gelijkaardig als dat van het vak Smart System. We werken terug agile. De grootste verschillen zijn dat er verschillende projecten zijn en dat er neer nadruk is op het draadloze aspect. Deze project opdrachten moeten met minimaal 3 worden uitgevoerd en maximaal 4 personen. De scope van het project is lineair met de groep grote.

Je mag ook een eigen project aanreiken in samenspraak met de docenten.
